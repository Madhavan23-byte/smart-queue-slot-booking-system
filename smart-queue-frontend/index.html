Key Features

Online Token Booking

Users can book tokens online without standing in physical queues.

Queue Tracking

Users can view current token, queue position, and waiting time.

Admin Control

Admin can call, complete, skip, or cancel tokens easily.

Crowd Indicator

The system shows low, medium, or high crowd level.

How It Works

1. Login

User logs into the system using email and password.

2. Book Token

User selects service, date, time slot, and books a token.

3. Track Queue

User checks token status and estimated waiting time.

4. Admin Updates

Admin manages the queue and calls the next token.
